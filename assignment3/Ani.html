JavaScript running different on browsers

becasue different browsers have slightly different versions of the Javaengine this means that the same javascript may run differently dependent on the browser.

Getting the size of the veiwport

The first example is how you decect the size of the veiw port on firefox and internet explorer internet explorer code: var myBrowserSize = [0, 0]; myBrowserSize[0] = document.documentElement.clientWidth; myBrowserSize[1] = document.documentElement.clientHeight; however this code will not work in fire fox however below is the code which will work in firefox: var myBrowserSize = [0, 0]; myBrowserSize[0] = window.innerWidth; myBrowserSize[1] = window.innerHeight;

Changing the CSS float property

document.getElementById("header").style.styleFloat = "left"; This code works when using internet explorer however if i try to run it in firefox it does not work. Below is the code which works in firefox: document.getElementById("header").style.cssFloat = "left";
