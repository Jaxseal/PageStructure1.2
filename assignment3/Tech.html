Navbars

Navbars are normally placed at the top of the page to help the user navigate between webpages how you create a navbar is by creating a list of items using the ul tag then you format the list so that the items are displayed horizontiatlly rather than vertically. Also the items in the lists are links to the other webpages so that when you click on then it will take you to the webpage you want to go to.
