Above is an example of a transition which allow for you to change the properties for a certain element.You do this by using the transition property.Below is the code which does this.The main difference between a transition and an animation is that a transition can only have two states where as animations can have multiple.

.tran {

width: 100px;

height: 100px;

background: purple;

transition: background 2s, transform 2s;

}

.tran:hover {

background: greenyellow;

transform: rotate(180deg);

}
