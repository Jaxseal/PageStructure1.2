Animation

Above is an example of an animation which allows to change how an element changes between styles. There are a list of properties for animation such as; keyframes, name, duration, delay, iteration count, direction, timing-function, fill-mode. You can use animations for multiple things one of which is if it doesnt repeat you how ever on this page i have set the animation-iteration-count to infinite which means that the animation will iterate forever how ever but you can put it to something such as 3 which means that it will only iterate three times.

@keyframes is where you put the styling for the elements which will be used during the animation here is an example:

@keyframes example {

0% {background-color: purple; left:0px; top:0px;}

25% {background-color: plum; left:0px; top:200px;}

50% {background-color: rgb(110, 197, 110); left:200px; top:200px;}

75% {background-color: palevioletred; left:200px; top:0px;}

100% {background-color: purple; left:0px; top:0px;}

 The percentage determines what the style of the element at a certain time during the animation. below is the code which describes how the features of the animation such as ho wmany times it will Itterate, how long it lasts, the name etc.

.ani{

width: 100px;

height: 100px;

position: relative;

background-color: plum;

animation-name: example;

animation-duration: 2s;

animation-iteration-count: infinite;

}
